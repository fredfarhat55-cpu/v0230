# Apex AI Assistant

A streamlined Next.js application with AI capabilities, reduced from 555+ files to under 100 files for better maintainability.

## Project Structure

- `app/` - Next.js pages and API routes
- `components/` - React components (UI and core apex components)
- `lib/` - Core utilities and APIs
- `hooks/` - React hooks
- `services/` - Service files

## Getting Started

1. Install dependencies:
```bash
pnpm install
```

2. Run the development server:
```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
pnpm build
pnpm start
```

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Radix UI components
